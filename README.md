# weather_diary
하루의 날씨와 일기를 기록할 수 있는 날씨 일기 Rest API

## 사용 기술 스택
* Spring Boot
* Spring JPA
* MySQL
* LogBack
* Swagger

## 구현 기능
* 날짜를 선택하여 날씨와 일기를 저장
* 선택한 날짜의 모든 일기 데이터 조회
* 선택한 기간중의 모든 일기 데이터 조회
* 날짜를 선택하여 일기 내용 수정
* 날짜를 선택하여 저장된 일기 삭제
* 매일 정해진 시간에 날씨 정보를 DB에 저장
* 예외 혹은 에러 발생시 로그 기록
* API Document에서 Rest API의 명세 확인
